Overview
This project is a simple calculator implemented in Python using the Tkinter library.
It's an excellent project for beginners to showcase their Python skills. 
The calculator features a basic user interface with buttons for arithmetic operations and a display area.
Users with basic knowledge of Python can easily understand and create this project, 
making it a great starting point for demonstrating Python proficiency.

Key Features
Function-Based: The calculator functions are implemented as separate functions,
making the code modular and easy to understand.

Tkinter UI: The project uses Tkinter for creating a graphical user interface with buttons,
making it visually resemble a real calculator.

Basic Arithmetic Operations: The calculator supports common arithmetic operations,
including addition, subtraction, multiplication, and division.

User-Friendly Design: The UI includes padding and layout considerations to enhance,
the calculator's appearance and usability.
